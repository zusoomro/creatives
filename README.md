# Creatives

## Overview

A personal project of mine, Creatives is a social media website where contractors such as DJ's and photographers can market themselves and connect with other creatives.

## Features

- Create an account
- Create a profile
- Make posts
- Like and unlike posts
- Make comments
- Update your profile
